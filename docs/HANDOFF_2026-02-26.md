## Core Pipeline (Production)
Entrypoint:
- `mosaic_module.main.create_temporal_mosaic(...)`

High-level method:
1. Build planar AOI square in EPSG:3857 with `aoi_size_km`.
2. Load `COPERNICUS/S2_HARMONIZED`, join `COPERNICUS/S2_CLOUD_PROBABILITY`.
3. Build cloud mask from s2cloudless only:
   - `cloud_prob >= 65` (default)
   - morphology cleanup + minimum cloud component size filter (`min_cloud_px=4`)
4. Build `clear_mask`.
5. Score scenes by AOI clear fraction only.
6. Keep top candidate scenes (`top_n_scenes`, default 8).
7. SNIC segmentation on cloud-masked reference image.
8. Assign each cluster to best scene if clear fraction >= `clear_thresh` (default 0.8).
9. Hierarchical fallback assignment on finer projected patch grids.
10. Stitch 13 bands and feather seams.
11. Rescue pass for remaining gaps:
   - use top rescue candidates (`top_n_scenes_rescue`, default 10)
   - fill from per-pixel minimum `cloud_prob` source among rescue candidates
12. Export:
   - full product to Drive
   - optional RGB product (`*_rgb`) to same Drive folder

Stage-1 extraction path:
1. Sample 5,000 Europe polygons with size stratification.
2. Assign each polygon to all overlapping 2.56 km global chips.
3. Export one exact 256×256 chip mosaic per unique chip to Drive folder `solar_openEO_stage1_v3`.
4. Download completed chip mosaics locally.
5. Extract `_image.tif`, `_mask.tif`, and `_quickview.png` per chip.
6. Package valid chips into `outputs/stage1/stage1_positives.h5`.

## Cloud Mask Method (Current)
File:
- `mosaic_module/cloud_mask.py`

Method:
- Cloud source: s2cloudless `cloud_prob` only (single-source policy).
- Default threshold: `cloud_prob_th = 65`.
- Morphology: `focal_min(1)` then `focal_max(radius=2 px)`.
- Speckle suppression: connected-component cloud size filter with default `min_cloud_px = 4`.
- Shadow stage: optional (`use_shadow=False` default).

Reason for this setup:
- Reduces single-pixel noise and permanent false-cloud artifacts.
- Keeps cloud policy simple and consistent across regions.

## Default Tuning (As Implemented)
- `clear_thresh = 0.8`
- `top_n_scenes = 8`
- `top_n_scenes_rescue = 10`
- `use_shadow_mask = False`
- `cloud_prob_th = 65` (inside `compute_clear_mask`)
- `min_cloud_px = 4` (inside `compute_clear_mask`)
